# Danksagungen / Lizenzen (3D, Fonts, Fotos)

Die **Vorschau-Geometrie** ist ein einfacher Quader im eigenen Code (kein externes Mesh). Spätere **glTF-Modelle** hier mit Autor und Lizenz eintragen.

Hinweis: Bevorzuge **CC0** oder **CC-BY** ohne **NC**, wenn der Konfigurator kommerziell genutzt wird.
